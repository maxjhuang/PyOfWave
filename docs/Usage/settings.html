Settings in PyOfWave Server¶

This file documents all the values in settings.py and protocols/__init__.py you can customise.

General settings¶

DOMAIN¶

Required (pyofwave.info for testing)

This value represents your domain name. This is used by the protocols to identify documents and users belonging to you. Should be a string.

DELTA_OBSERVER_PROCESSES¶

Default: Your CPU count

The number of processes to launch to handle changes in documents “deltas”. Should be a integer.

DELTA_OBSERVER_TIMEOUT¶

Default: None

The number of tasks a process should run before it times out. Should be a integer.

Backing stores¶

If you wish to customize how data is handled in PyOfWave, use these variables. If you need more details, please see CoreModule.

DATASOURCE_STORAGE¶

Default: ‘storage.files.FileStorage’

The main storage object used by all protocols and extensions. Use a dotted path that points to the module.

protocols/__init__.py¶

Note

These settings have to be moved out of the package to allow user reconfiguration.

Perhaps add a ADDONS setting.

In this file, you setup any changes to the protocols you have in your runtime. Instructions for doing this can be found at Protocols.
