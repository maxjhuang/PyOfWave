Installing PyOfWave on Your Server¶

This file leads you through how to install PyOfWave on your server.

Setting Up the Server¶

  1. Download PyOfWave at https://github.com/pyofwave/PyOfWave/ .

  2. From inside the downloaded and uncompressed directory:

    python setup.py install

  3. Create a file named settings.py as described by Settings in PyOfWave Server and from it’s directory run:

    pyofwave

Note

Should we add a argument of the file to import?

Setting Up the Client¶

  1. Ensure that the HTTP server you plan to use supports:
    • Serving raw files.
    • Supports Python CGI.
  2. Customize pyofwave_client/index.html and pofwave_client/pyofwave_client/pyofwave_client.html to include your branding.
  3. Place these files in your server’s web directory.
  4. Start your HTTP server.
